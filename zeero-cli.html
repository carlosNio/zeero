ZCLI - Zeero Command Line Interface

Esse Componente é de grande uso em desenvolvimento, já que as suas funcionalides agilizam o trabalho. Operações tais como: criação de arquivos , migração de dados ou tabelas, execução de tarefas tendem a gastar tempo quando feito manualmente, por isso é recomandado o uso do Zcli.


O Zcli possuí um arquivo na raíz do projecto ( zcli ) que serve de script para o php, com qual poderemos executar os nossos comandos.


  • Estrutura de um comando

    • > php zcli [NOME_DO_COMANDO] [:ARGUMENTO] [VALOR] [OPÇÂO 1] [OPÇÂO 2] ... [OPÇÂO N]

  • Lista de Comandos Disponíveis

    • Nome Usado Para
    MAKE Criação de arquivos.
    MIGRATE Migração de tabelas á Base de Dados.
    SECURITY Actualização de items de segurança
    SERVE Inicialização do servidor de desenvolvimeto local.

  • Exemplos do Comando MAKE

    • > php zcli make:controller Post
    > php zcli make:controller Post -- r // REST rsource
    > php zcli make:model Comment
    > php zcli make:migration Post
    > php zcli make:request Post
    > php zcli make:seed Post
    > php zcli make:console_command MyCmd

  • Exemplos do Comando MIGRATE

    • > php zcli migrate User --up // create or edit a table
    > php zcli migrate User --down // drop a table
    > php zcli migrate User --up --m // create a table and generate the model class
    > php zcli migrate:all --up // create or edit all migrations tables
    > php zcli migrate:all --down // drop all migrations tables

  • Exemplos do Comando SECURITY

    • > php zcli security:key generate // Generate a new Application KEY

  • Exemplos do Comando ROUTE

    • > php zcli route:list // view all routes

  • Exemplos do Comando SERVE

    • > php zcli serve // start localhost ( default por:1000 )
    > php zcli serve 5000 // start localhost at port: 5000

  • Criando Comandos

    • Os passos para a criação de comandos zcli para uma aplicação são:


    • Criar uma classe Comando , via comando MAKE, a classe será criada na Pasta Controllers/Console.
    • Registar o Comando criado na classe Zcli, via metódo estático registerCommand(), este registro deve ser feito no arquivo console.php dessa mesma Pasta.


    Registrando o Comando no arquivo console.php do directório Controllers/Console 
                                
    Zcli::registerCommand('myCmd' , MyCommand::class);
    > php zcli myCmd

    Adicionando Argumentos Ao Comando 
                                
    public static $_arguments = [ 'arg' ]; //registrar o argumento na propriedade da classe

    // ...

    // criando o metodo
    public function arg()
    {
      if ($this->input_value)
        echo "hello {$this->input_value}";
      else
        echo "hello from :arg";
    }
    > php zcli myCmd:arg // hello from :arg
    > php zcli myCmd:arg carlos // hello carlos
