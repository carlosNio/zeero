Iniciar um Projecto com a Framework

1º PASSO: Ferramentas

Para iniciar um novo projecto é recomendado ter instalado na máquina os seguintes items:

  • PHP 7.4+ .
  • Composer .
  • Visual Studio Code .

2º PASSO: Instalação

Crie uma pasta com o nome do novo projecto e abra o terminal ( no visual studio code use a combinação CTRL + SHIFT + C ), e execute o seguinte comando:

> composer create-project zeero-fw/app-start ./

3º PASSO: Hello World

Após instalado na sua máquina, para rodar o projecto abra o terminal e execute o seguinte comando:

> php zcli serve

Após a execução desse comando o projecto será iniciado na porta 8080 do localhost, então abra o seu navegador e insira a URL localhost:8080 para ver a mensagem de boas vindas.
