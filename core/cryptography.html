FACADES HELPERS CRYPTO
  • Zeero\Core\Crypto\Crypto

    • Essa Classe contém duas finalidades: criptografar e decriptografar textos e arquivos.

    O Metódo de crifra padrão é o AES-256-CBC, e a chave utilizada é a definida no arquivo .env no item APP_KEY. 

            
    // encrypt text
    $encrypted = Crypto::encryptText('message to send');
        
    // decrypt text
    $decrypted = Crypto::decryptText($encrypted);
        
    // change the cipher method 
    //throws InvalidArgumentException if the cipher method is not in openssl_get_cipher_methods() array 
    Crypto::setMethod('cipher method');
        
    // change the key   
    Crypto::setKey('my secret key');
     
    // encrypt file
    // file.json.enc if success , throws NotFoundException if the file not exists 
    $encrypted_filename = Crypto::encryptFile('file.json' , 'file.json.enc');
    
    // decrypt file
    // file.json.dec if success 
    $decrypted_filename = Crypto::encryptFile($encrypted_filename , 'file.json.dec');

  • Zeero\Core\Crypto\Hashing

    • Essa classe tem duas finalidades: gerar token ou id randômica , gerar e verificar hash ( via password_hash() ). 

          
    $bytes = 6;
    $token = Hashing::random_token($bytes);
    
    $id = Hashing::random_id();
     
    // generate a hash with BCRYPT ALGORITHM ( default id: 2) 
    $password = Hashing::hash('1234321');
       
    // generate a hash with PASSWORD_DEFAULT ALGORITHM ( id: 1 ) 
    $password = Hashing::hash('1234321' , 1);
    
    // generate a hash with BCRYPT ALGORITHM and COST 13 ( default cost: 11) 
    $password = Hashing::hash('1234321' , 2 , 13);
     
    // verify a hashed password
    $hashed = '$2y$11$UuCehXFyGvP/h.xyV26AgeN4WiynN4KNnliJS53nMznAKYOo5Osgq';
    $ok = Hashing::verify_hash('1234321' , $hashed);
